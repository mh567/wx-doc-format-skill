# WX 文档格式转换

将 Markdown 或 DOCX 文档转换为模板驱动的 WX 格式 `.docx` 文件。

## 核心功能

- 理解文档语义，自动识别标题层级、列表、表格、题注，源文档格式不论多乱都能正确解析
- 自动重写编号和样式，清除手工序号，统一套用模板格式
- 图片、表格、题注完整保留，自动编号，无样式残留
- 输出审计报告，样式不合规、结构风险一目了然
